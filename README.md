# REST API Справочник Организаций

API для работы со справочником организаций, зданий и видов деятельности.

## Стэк технологий
- **FastAPI**: Веб-фреймворк
- **SQLAlchemy**: ORM для работы с БД
- **Alembic**: Миграции базы данных
- **PostgreSQL**: База данных
- **Pytest**: Тестирование

## Быстрый запуск

### 1. Запуск через Docker Compose
```bash
docker-compose up --build
```
API будет доступно по адресу: http://localhost:8000

### 2. Заполнение тестовыми данными
При первом запуске база пуста. Для заполнения выполните:
```bash
docker-compose run --rm app python src/seed.py
```

### 3. Авторизация
Все методы (кроме `/`) защищены статическим API ключом.
- Заголовок: `X-API-Key`
- Значение по умолчанию: `test-api-key-123` (настраивается в `.env`)

### 4. Документация API
- **Swagger UI**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc

## Функционал
- Список организаций в конкретном здании.
- Список организаций по виду деятельности.
- Поиск организаций в радиусе или прямоугольной области.
- Получение информации об организации по ID.
- Поиск по дереву деятельностей (включая вложенные уровни до 3).
- Поиск организации по названию.

## Тестирование
Для запуска тестов используйте:
```bash
docker-compose run --rm app pytest tests/test_api.py
```
